# Scratch Workspace

Ephemeral working files generated during development tasks.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `analysis/` | Codebase investigations, pattern analysis |
| `notes/` | Brainstorming, option exploration |
| `generated/` | Test data, mock responses |

## Lifecycle

- Tracked in git (visible in `git status`)
- **Delete after task/session completes**
- If content should persist, move to `.plan/` or `docs/`

See `AGENTS.md` for full specification.
